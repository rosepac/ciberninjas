Bootstrap es la opción de marco CSS más popular cuando se trata de un desarrollo web front-end receptivo y móvil. Además, es gratuito y de código abierto.

Lanzado en 2011, Bootstrap tiene todas las plantillas de diseño basadas en CSS y JS que se requieren para crear botones, formularios, navegación, tipografía y otros componentes de la interfaz.

Principales preguntas de la entrevista de Bootstrap
Existe una gran posibilidad de que una entrevista de trabajo de desarrollo web incluya preguntas basadas en Bootstrap. Por lo tanto, debe mantener su preparación de Bootstrap para obtener el papel. Para ayudarlo con la causa, aquí están las principales preguntas de la entrevista de Bootstrap con respuestas:

## **¿Qué quiere decir Bootstrap Grid System?**

Bootstrap Grid System es un sistema receptivo, primero móvil que escala hasta 12 columnas según el aumento en el tamaño del dispositivo o la ventana gráfica. El sistema cuenta con clases predefinidas para opciones de diseño fáciles y combinaciones poderosas para generar diseños semánticos de manera efectiva.

## **¿Qué es Normalize en Bootstrap?**

Para establecer la coherencia entre navegadores, Bootstrap utiliza Normalize. Es un pequeño archivo CSS capaz de ofrecer una mejor coherencia entre navegadores en el estilo predeterminado de los elementos HTML. Además, Normalize.css es una alternativa moderna y lista para HTML5 a los restablecimientos de CSS .

Pregunta:  ¿Qué entiende por la barra de navegación Bootstrap? ¿Cómo vas a crear uno?
Respuesta : Una de las características más destacadas de Bootstrap, una barra de navegación es un componente 'meta' sensible que sirve como encabezado de navegación para una aplicación o sitio web. Puede haber varias barras de navegación en una aplicación o sitio web.

En las vistas móviles, una barra de navegación se colapsa y se vuelve horizontal cuando aumenta el ancho de la ventana gráfica disponible. La barra de navegación incluye estilos para la navegación básica y los nombres de los sitios. A continuación se explica cómo crear una barra de navegación en Bootstrap:

Agregue las clases .navbar y .navbar-default a la etiqueta <nav>
A continuación, agregue role = "navigation" al elemento <nav>
Ahora, agregue .navbar-header, una clase de encabezado, al elemento <div>. Para hacer el texto un poco más grande, incluya un elemento <a> con la clase navbar-brand
Agregue una lista desordenada con las clases .nav y .navbar-nav para agregar enlaces a la barra de navegación Bootstrap
Pregunta:  ¿Puede explicar la ruta de navegación de Bootstrap?
Respuesta : Una ruta de navegación de Bootstrap es una excelente manera de mostrar información basada en jerarquías para un sitio web. Simplemente, es una lista desordenada con una clase .breadcrumb. CSS agrega automáticamente el separador para una ruta de navegación de Bootstrap.

En los blogs, la ruta de navegación puede mostrar categorías, fechas de publicación o etiquetas. Indica la ubicación de la página actual dentro de una jerarquía de navegación.

Puede agregar cualquier tipo de contenido HTML, como botones, encabezados o párrafos, en miniaturas. Así es como personalizar las miniaturas usando Bootstrap:

Cambie la etiqueta <a> con la clase .thumbnail a una etiqueta <div>

Nota : - Si desea agrupar varias imágenes, colóquelas en una lista desordenada. Cada elemento de la lista flotará hacia la izquierda.

Cree un contenedor <div> y agregue una clase .alert y una de las 4 clases contextuales, a saber, .alert-danger, .alert-info, .alert-success y .alert-warning, para agregar una alerta básica
Agregue la clase opcional .alert-descartable al <div>
A continuación, agregue un botón de cierre
Finalmente, use el elemento <button> con el atributo de datos data-dispats = "alert"

Agrega un <div> con una clase .progress

Agregue un <div> con las clases .progress y .progress-striped

Ahora, agregue un atributo de estilo con el ancho expresado como un porcentaje, por ejemplo, estilo = "70%"

El objetivo principal de un objeto multimedia Bootstrap es hacer que el código necesario para desarrollar bloques de información sea increíblemente más pequeño. Para lograr una fácil extensibilidad, un marcado ligero y otros aspectos deseables, se aplican clases a algunos de los marcados simples.

Pregunta: ¿Para qué sirven las clases .media y .useful en Bootstrap?

Pregunta:  ¿Qué quieres decir con Bootstrap bien?

Pregunta:  ¿Por qué usamos el complemento de afijo en Bootstrap?
Respuesta : Usamos el complemento de afijo en Bootstrap para agregar un <div> a una ubicación determinada en una página web. El complemento también permite activar y desactivar el anclaje del <div> adjunto. Los íconos sociales son el ejemplo más popular del uso del complemento de afijo en Bootstrap.

Nota : - El <div> adjunto comienza desde una ubicación particular en la página web y se desplaza con él. Sin embargo, después de una determinada marca, se bloqueará en su lugar, lo que dejará de desplazarse con el resto de la página web.

Pregunta: ¿Cuál es la necesidad de Bootstrap?
Respuesta: Bootstrap actúa como un marco front-end que admite el desarrollo de varios tipos de aplicaciones web como HTML, JS y CSS.

Pregunta: ¿Cuál es el beneficio de Bootstrap?
Respuesta: Ayuda a desarrollar un diseño altamente receptivo, rápido y fácil de usar. La principal ventaja de Bootstrap se observa al desarrollar aplicaciones móviles que tienen plantillas de diseño. Estas plantillas ayudan a crear la interfaz de usuario, incluida la pestaña de alerta, el menú desplegable, los formularios, etc.

Pregunta: ¿Cuáles son los componentes clave de Bootstrap? ¿Explique?
Respuesta: Los componentes clave de Bootstrap incluyen lo siguiente:

Andamio: admite el sistema de cuadrícula, estilos y fondos de varios tipos.
CSS: ofrece archivos CSS para desarrollar aplicaciones web.
Personalizar: apoya el desarrollo de la personalización del marco.
Complementos JS: consta de complementos JS y JQuery necesarios en el proceso de diseño y aplicación web.
Pregunta: ¿Qué es un contenedor Bootstrap?
Respuesta: Un contenedor Bootstrap consta de código HTML que se puede colocar para convertirlo en una aplicación web rápida y con gran capacidad de respuesta.

Pregunta: ¿Definir tipos de diseño en Bootstrap? ¿Explique?
Respuesta: Hay dos tipos de diseño en Bootstrap que incluyen diseño fijo y diseño fluido. El diseño fluido ayuda a crear una pantalla de ancho completo que se ajusta al tamaño del navegador. El diseño fijo, por otro lado, no puede ajustarse a sí mismo según la pantalla del navegador y tiene un diseño fijo de 940 px.

Pregunta: ¿Podemos mostrar código en Bootstrap? ¿Cómo?
Respuesta: Sí, podemos mostrar el código en Bootstrap. Se puede mostrar mediante etiquetas como la etiqueta <code> y la etiqueta <pre>. La etiqueta <code> puede ayudar a mostrar el código en línea, mientras que la etiqueta <pre> puede ayudar a mostrar el código con diferentes líneas.

Pregunta: ¿Definir alertas de Bootstrap?
Respuesta: Las alertas de Bootstrap están destinadas a crear mensajes de alerta al agregar estilos y hacer que se vean lo suficientemente atractivos como para captar la atención del espectador.

Pregunta: ¿Definir miniaturas de Bootstrap?
Respuesta: Las miniaturas de Bootstrap ayudan a usar las imágenes de diseño, el texto, los videos y otras funciones en un sistema de cuadrícula. De esta forma, podemos agregar varias miniaturas mediante etiquetas alrededor de la imagen.

Pregunta: ¿Qué pasará si agregamos muchas etiquetas con la clase de miniaturas alrededor de la imagen?
Respuesta: Esto dará como resultado la creación de cuatro píxeles de relleno y también desarrollará un borde gris.

Pregunta: ¿Cuáles son las etiquetas en Bootstrap?
Respuesta: Las etiquetas en Bootstrap se utilizan para ofrecer sugerencias, recuentos y otra información que podría ayudar a marcar la página web.

Pregunta: ¿Cuál es el beneficio de Jumbotron en Bootstrap?
Respuesta: Jumbotron en Bootstrap se puede usar para aumentar el tamaño del encabezado y aumentar los márgenes en la página de destino.

Pregunta: ¿Qué se normaliza en Bootstrap?
Respuesta: Normalizado en Bootstrap es un pequeño archivo CSS. Se utiliza para crear coherencia entre navegadores.

Pregunta: ¿Definir paneles en Bootstrap?
Respuesta: Los paneles en Bootstrap son diferentes componentes que se pueden colocar en el componente DOM en una caja con el propósito de recuperar el panel básico usando el class.panel como el elemento <div>.

Pregunta: ¿Qué es un sistema de cuadrícula en Bootstrap?
Respuesta: Un sistema de cuadrícula en Bootstrap ayuda a crear hasta 12 columnas en una página web. De esta forma, ayuda a colocar los contenidos de la página web en diferentes columnas, que pueden usarse como títulos o subtítulos.

Pregunta: ¿Hay clases de cuadrícula en Bootstrap?
Respuesta: Sí, hay cuatro clases de cuadrícula en Bootstrap. Estos incluyen xs, sm, md y lg.

Pregunta: ¿Cuál es la función de la clase de cuadrícula xs?
Respuesta: La clase de cuadrícula xs se usa para pantallas de teléfonos móviles, que tienen menos de 786 px de ancho.

Pregunta: ¿Cuál es la función de la clase sm grid?
Respuesta: La clase de cuadrícula sm se usa para las pantallas de tableta anchas, que tienen más de 786 px de ancho.

Pregunta: ¿Cuál es la función de la clase md grid?
Respuesta: La clase de cuadrícula md se usa para la pantalla de computadoras portátiles pequeñas, que tiene un ancho igual o superior a 992 px.

Pregunta: ¿Cuál es la función de la clase LG grid?
Respuesta: La clase de cuadrícula LG se usa para pantallas más grandes de computadoras portátiles y de escritorio, que son iguales o incluso mayores que 1200 px de ancho.

Pregunta: ¿Qué es un estilo global? ¿Se utilizan en Bootstrap?
Respuesta: El estilo global se refiere a la versión estandarizada del tamaño de fuente, la altura de la línea y otras características que se utilizan y son aceptables en la plataforma internacional. Se utilizan en Bootstrap para Bootstrap Default Typography.

Pregunta: ¿Cuál es el estilo global de Bootstrap Default Typography?
Respuesta: El estilo global para Bootstrap Default Typography incluye lo siguiente predeterminado:

El tamaño de fuente de 14 px
La altura de la línea de 1.428.
El estilo de fuente de Helvetica y Arial con respaldo sans-serif
Todos los estilos, el tamaño y la altura mencionados anteriormente deben usarse en el cuerpo y en los párrafos.

Pregunta: ¿Existe algún requisito previo para ejecutar Bootstrap correctamente?
Respuesta: Sí, existe un requisito previo de JQuery, que es necesario para ejecutar Bootstrap correctamente. Actúa como los únicos complementos de JavaScript en Bootstrap.

Pregunta: ¿Hay alguna diferencia entre Foundation y Bootstrap?
Respuesta: Sí, existe una diferencia entre Foundation y Bootstrap. The Foundation es compatible con los procesadores SASS y se utiliza principalmente para el diseño de interfaces de usuario móviles. Por otro lado, Bootstrap admite el diseño de portales web móviles y de escritorio que utilizan preprocesadores menores.

Pregunta: ¿Conoce el complemento de transición en Bootstrap? ¿Por qué se usa?
Respuesta: Sí, conozco el complemento de transición en Boostrap. Esto se usa para proporcionar efectos de transición simples. Estos incluyen desvanecimiento o incluso deslizamiento en modales.

Pregunta: ¿Qué es un complemento Affix y cómo ayuda a Bootstrap?
Respuesta: El complemento Affix se utiliza como un ícono social en una página web. Permite que el <div> se adjunte a una ubicación particular de la página. Comenzará en una ubicación, pero cuando la página llegue, se alejará y obtendrá un bloque con el <div> en su lugar. De esta manera, ayuda a desplazarse por el resto de la página web.

Pregunta: ¿Qué es el complemento scrollspy en Bootstrap?
Respuesta: El complemento Scrollspy en Bootstrap admite opciones de complemento de navegación de actualización automática para aplicar.
