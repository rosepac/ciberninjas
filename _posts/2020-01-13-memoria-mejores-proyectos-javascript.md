* Los proyectos más populares del año
* Frameworks Front-End
* Frameworks Node.js
* Ecosistema React
* Ecosistema Vue
* Ecosistema Angular
* Librerías de Desarrollo Móvil
* Compiladores JS
* Librerías de Montaje
* Bibliotecas de Testing
* Editores de trabajo y picado de código
* CSS en Javascript
